# KB-v3 Werkcommando's

## Push naar Shopify
shopify theme push --theme 156048785573 --store kwispelbox.myshopify.com

## Pull (ALLEEN settings, nooit zonder --only)
shopify theme pull --only config/settings_data.json --theme 156048785573 --store kwispelbox.myshopify.com

## Theme check
shopify theme check

## Git
git add -A && git commit -m "bericht" && git push
